# Claude AI Chatbot

A simple AI chatbot built using the **Anthropic Claude API** and Python.  
This project demonstrates how to connect to the Claude API, send prompts, and receive responses using a lightweight Python implementation.

## Features

- Connects to Anthropic Claude models via API
- Uses environment variables to securely store API keys
- Simple chatbot interaction through Python
- Built and tested in a Jupyter Notebook

## Tech Stack

- Python
- Anthropic API
- python-dotenv
- Jupyter Notebook
